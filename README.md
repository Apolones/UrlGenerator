## Installation

1. **Clone the repository**:
    ```bash
    git clone https://github.com/Apolones/UrlGenerator.git
    cd UrlGenerator
    ```

2. **Setup**:
 Configure the properties in:
     `src/main/resources/application.properties`

3. **Run**:
    ```bash
    mvn clean install
    ```
    ```bash
    mvn spring-boot:run
    ```
